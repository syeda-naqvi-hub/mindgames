# MindPlay 🧠✨

## Interactive Web App Features:
- **Quiz Game** - Multiple choice questions with progress tracking
- **Personality Test** - 1-5 scale assessment with results
- **Calculator** - Scientific calculator with currency converter
- **Responsive Design** - Mobile-first approach
- **Dark Mode** - System preference support
- **Smooth Animations** - Modern UI/UX with hover effects

## Quick Start 🚀
1. Open `index.html` in any modern browser
2. No build tools required - Pure HTML/CSS/JS
3. All features work offline (except real currency API if added)

## Pages 📄
- `index.html` - Home
- `about.html` - About
- `pages/quiz.html` - Interactive Quiz
- `pages/personality.html` - Personality Assessment  
- `pages/calculator.html` - Calculator + Currency Converter

## Features Implemented:
✅ **Fully functional quiz** with scoring  
✅ **Personality test** with results  
✅ **Working calculator** (add, subtract, multiply, divide)  
✅ **Currency converter** (mock data - easy to connect real API)  
✅ **Responsive navbar** with active states  
✅ **Modern gradient UI** with glassmorphism  
✅ **Progress bars** and animations  
✅ **Sound effects** (optional - add MP3 files)  

## Customization 🔧
- Add more quiz questions in `data/quizData.js`
- Update personality logic in `js/personality.js`
- Connect real currency API in `js/currency.js`
- Add images to `assets/images/`
- Include sound files in `assets/sounds/click.mp3`

## Tech Stack 🛠️
```
HTML5 | CSS3 | Vanilla JavaScript
Responsive Design | CSS Grid/Flexbox
Modern CSS (Gradients, Backdrop-filter)
```
